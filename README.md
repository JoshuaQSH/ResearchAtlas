## Notes On This Initial Pass
- The first run favors field-defining papers and widely adopted systems papers over exhaustive per-topic coverage.
- arXiv is used when a paper is already highly visible and the conference version is not easier to cite consistently.
- Citation analytics are intentionally conservative in this pass; `AUTHORS&GROUPS.md` leaves metrics as `N/A` unless a profile was explicitly verified.
