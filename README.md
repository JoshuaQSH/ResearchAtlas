# ResearchAtlas

ResearchAtlas is a curated paper atlas for systems, infrastructure, and algorithms behind modern AI. The repository is organized by year and broad technical area, with an expanding survey starting at 2022 and a venue-coverage backfill for adjacent systems and software-engineering conferences.

## Notes On Current Coverage
- All topics in `TOPICS.md` are now represented at least once, including `Explainable AI`.
- `Hodgepodge` is used for venue-coverage entries that are only weakly aligned with the atlas core topics.
- arXiv is still used when a paper is already highly visible and the conference version is not easier to cite consistently.
- Citation analytics remain conservative in this pass; `AUTHORS&GROUPS.md` still leaves metrics as `N/A` unless a profile was explicitly verified.
