# ResearchAtlas

ResearchAtlas is a curated paper atlas for systems, infrastructure, and algorithms behind modern AI. The repository is organized by year and broad technical area, with a venue-coverage pass that now represents every conference listed in `COFERENCES.md`.

## Notes On Current Coverage
- All topics in `TOPICS.md` are represented at least once.
- Every conference listed in `COFERENCES.md` is represented by at least one paper in the repository.
- `Hodgepodge` is used for venue-coverage entries that are only weakly aligned with the atlas core topics.
- `CATALOGUA.md` is generated from the tracked paper corpus and currently records `Affiliations: N/A` when affiliation data has not been explicitly verified.
