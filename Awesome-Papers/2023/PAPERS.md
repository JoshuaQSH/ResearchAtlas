# Awesome Papers 2023

## [vLLM: Easy, Fast, and Cheap LLM Serving with PagedAttention]

- **Why included:** The serving system that normalized KV-cache virtualization as a baseline idea.
- **Venue:** SOSP 2023
- **Paper Link:** https://arxiv.org/abs/2309.06180

## [Llama 2: Open Foundation and Fine-Tuned Chat Models]

- **Why included:** A field-defining open-weight model release that influenced downstream systems and compression work.
- **Venue:** arXiv / Meta AI
- **Paper Link:** https://arxiv.org/abs/2307.09288

## [SmoothQuant: Accurate and Efficient Post-Training Quantization for Large Language Models]

- **Why included:** One of the core post-training quantization references for practical LLM deployment.
- **Venue:** ICML 2023
- **Paper Link:** https://proceedings.mlr.press/v202/xiao23c.html
