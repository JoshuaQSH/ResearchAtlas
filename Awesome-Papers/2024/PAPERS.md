# Awesome Papers 2024

## [SGLang: Efficient Execution of Structured Language Model Programs]

- **Why included:** Strong execution model for structured LLM applications and high-throughput serving.
- **Venue:** NeurIPS 2024
- **Paper Link:** https://arxiv.org/abs/2312.07104

## [FlashAttention-2: Faster Attention with Better Parallelism and Work Partitioning]

- **Why included:** Major second-generation attention kernel paper with broad adoption.
- **Venue:** ICLR 2024
- **Paper Link:** https://openreview.net/forum?id=mZn2Xyh9Ec

## [QServe: W4A8KV4 Quantization and System Co-design for Efficient LLM Serving]

- **Why included:** Representative system-and-kernel co-design paper for low-bit LLM serving.
- **Venue:** 2024 preprint
- **Paper Link:** https://arxiv.org/abs/2405.04532
