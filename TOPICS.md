# Topics

The atlas currently tracks papers in the following topic buckets. Papers may carry multiple tags when a contribution spans systems, algorithms, and infrastructure.

## Distributed Deep Learning
- Parallel training systems, sharding, MoE routing, pipeline and tensor parallelism.

## Graph Neural Networks
- Graph representation learning, graph transformers, scalable GNN systems, and graph explainability.

## Explainable AI
- Interpretable model behavior, attribution methods, faithful explanation, and explanation-aware model design.

## Sparse Matrix & Kernels
- Sparse computation, sparse tensor formats, sparse attention, and sparse linear algebra kernels.

## Compiler Optimization
- Program scheduling, autotuning, kernel generation, IR design, and graph lowering for ML workloads.

## Model Compression
- Pruning, distillation, low-rank adaptation, and compressed inference pipelines.

## Model Quantization
- Post-training quantization, quantization-aware training, low-bit kernels, and calibration methods.

## LLM Systems & Algorithms
- Large language model serving, decoding, reasoning, long-context methods, and inference acceleration.

## AI Infrastructure
- End-to-end training and serving stacks, deployment frameworks, memory management, and cluster-scale infrastructure.

## MLIR
- MLIR-based compilation, dialect design, and compiler pipelines for ML workloads.

## CUDA Kernels
- CUDA-level optimization, fused kernels, attention kernels, and Tensor Core utilization.

## GPU Optimization
- GPU occupancy, memory bandwidth, tiling, and hardware-aware acceleration for ML workloads.

## HPC
- High-performance computing methods, large-scale communication, and performance portability for AI systems.

## Hodgepodge
- Papers included to satisfy venue coverage when they are only weakly aligned with the atlas core topics; prefer AI-adjacent software, systems, or compiler work.
